Flow map is a texture containing a unique 2D flow vectors for every point on the surface.

You can use these editor to paint your own flow map, please google for more infos.
Flow field editor - http://algoholic.eu/flow-field-editor-still-alive/
FlowMap Painter - http://teckartist.com/?p=96

You can use "Advance Shader Control" options to control the flow speed and strength.
